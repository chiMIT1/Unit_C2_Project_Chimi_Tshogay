Lesson 1: HTML

HTML

HTML stands for Hyper Text Markup Language, the language that makes the World Wide Web possible.

HTML was invented by Tim Berners-Lee in 1989. A markup language formats text, such that when displayed, the markups do not appear. In HTML, the markups are called tags, and are shown with the < and > symbols, which we call angle brackets.

Writing HTML code

HTML code can be written in any text editing program.

On Windows, you can edit HTML using the Notepad program.

On the Mac, you can edit HTML using the TextEdit program, configured for plain text editing.

For Chromebooks, there are not a lot of good options for text editors capable of creating HTML files.

Essential HTML elements

A properly written web page should have the following elements:

  • The HTML5 doctype declaration.
  • The html tag begins and ends the page.
  • The head tag holds the page's metadata.
  • The title tag gives the words that show up on web browser tabs.
  • The body tag contains the visible content of the page.
