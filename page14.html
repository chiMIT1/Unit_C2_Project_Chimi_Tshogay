Create an HTML Tutorial site

Your objective is to create an HTML tutorial website:

  • This site must contain at least eight web pages, including the home page (index.html), and at least seven other pages (page2.html through page8.html).
  • Each page must contain a lesson about writing HTML code, including showing what the code looks like, as well as the result of writing that code if visible on a web page.
  • The home page should have a link to each page, as well as a “start here” link to go to page2, which should start the lessons.

Have fun. I hope you enjoyed this tutorial.
