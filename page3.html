Types of Tags

HTML has two types of body elements: Block-level elements will automatically appear on separate lines on the page.

<p> and <h> are examples

Inline elements can appear next to one another horizontally, so that text with an inline element in it will not start a new line.

<strong> and <i>

List Elements

There are two types of list tags:

<ol> creates an ordered list (a list with numbers).

<ul>

creates an unordered list ( a list with bullets instead of numbers).

Both list types use <li> (list item) for the items within the lists

ol,ul, and li are block-level elemnents.
