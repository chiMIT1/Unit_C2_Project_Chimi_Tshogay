Lesson 6: Iframes

Iframe

Iframes allow one HTML document to display the contents of another HTML document or web page. We use the <iframe> element to accomplish this.

Example:

HTML iframe Syntax

<iframe src="url" title="description"></iframe>

The src attribute is used to specify the path to the HTML document or web page that will be displayed inside the iframe.

Iframes are inline elements, meaning if the width of an iframe is less than the width of the page, other content may appear to the right or left of it.

The title attribute is used to provide a description for screen readers (and in case the source page is inaccessible), describing the page that should be displayed within the iframe.

Height and Width

Although not strictly necessary, you can use the height and width attributes to control the size of the iframe window.

You can specify either height/width in terms of pixels (absolute values) or percentages (relative to the current window or parent element).

Example:

<iframe src="url" title="description" height="400" width="40%"></iframe>
