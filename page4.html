Lesson 3: Comments and Attributes

Comments

Comments are text hidden within an HTML document. In order to create an HTML comment, you must use the characters <!-- before the comment text and --> after it.

Attributes

Attributes are codes that appear within a tag that modify it in some way. The general syntax for an attribute is:

<tagName attributeName="value"> content </tagName>

Ex:

Links

Links use the anchor or <a> element to make a hyperlink to another page or to another location on the same page. It is an inline element. To locate the link, you need the href attribute.

The general syntax for a link is:

<a href="pathToFile">LinkText</a>

Ex:

More link attributes

download: lets the user download a file.

target: specifies a specific place within the page the link opens.

Some targets:

  • _blank: Opens in a new window or tab.
  • _top: Opens at the top of the page.
  • _self: Opens within the same iframe.
  • _parent: Opens within the parent of the current frame.

Embedding Images

Images are embedded on a web page using the <img> element. It is an empty element (no trailing tag) and works inline. To locate the image, you must use the src attribute. You must also use the alt attribute to describe the image for screen readers or if the image will not load.

Sizing elements

You can control the size of images by using either the width or height attributes, or both. The actual dimensions are specified in pixels.

If you only specify height or width, the image will scale proportionally. If you specify both, the image will fix itself to the dimensions you chose.
