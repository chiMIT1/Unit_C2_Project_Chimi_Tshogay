Lesson 4: Headings and Paragraphs

Headings

Headings are used to create an outline structure on a web page.

There are 6 heading types: <h1> through <h6>.

Aside from the size of the text, headings help assistive devices like screen readers.

All headings are block-level elements.

Paragraphs

The <p> tag is the workhorse of web design. Most blocks of text use paragraph tags for formatting.

Paragraphs are block-level elements, creating space above and below them on the page.

If you want to create a line break, use the <br> tag.

Horizontal Rule and Preformatted Text

If you want to create a line between blocks of text, use the <hr> element. The <hr> tag is an empty element.

If you want to preserve multiple spaces or line breaks (for text like poems), you can use the preformatted text <pre> tag. The <pre> tag is a block-level element.
