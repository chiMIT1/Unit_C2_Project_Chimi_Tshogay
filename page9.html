Lesson 8: Entities

Character Encoding Schemes

  • ASCII (American Standard Code for Information Interchange) - The first and most widely used character encoding system before Unicode.
  • EBCDIC (Extended Binary Coded Decimal Interchange Code) - Used by IBM mainframe computers until IBM converted to ASCII in the 1980s.
  • ANSI (American National Standards Institute) or Windows-1252 - An extended version of ASCII developed by ANSI and used in Windows until Unicode.
  • ISO-8859 - Developed by the International Organization for Standardization, it was a variety of formats for different languages that existed before Unicode.

Utilizing Unicode

Unicode has several character sets. The most popular for web browsers is Unicode Text Format 8-bit, or UTF-8.

To ensure browser compatibility for your web page, you can use the following code in the head section of your page:

<meta charset="utf-8">

HTML Entities

Several characters are reserved for use by HTML. Other characters are difficult to reproduce on the keyboard, or are things like emojis that can't be produced without special apps. To display these characters as text or emojis, you can use the following methods:

(Nickname method): & + nickname + ; Example: < becomes &lt;

(Number method): & + # + Unicode number + ; Example: < becomes &#60;
