Lesson 7: Head Elements

Recap: The Head

As we have learned, every web page has a head and a body. The head is where the page's metadata is found, along with everything else that is meant not to appear on the visible part of the page.

Example:

What Should Go in the Head?

The head section contains several important elements:

  • Title: The title is what shows up at the top of the web page window or tab.
  • Metadata: Data that is invisible to the viewer but visible to web crawlers, which is important for SEO and search results.
  • Styles: Instructions to the web browser on how the web page should look (CSS styles).
  • Scripts: JavaScript code that must run before the body of the web page loads.
  • Links: Important files that are linked to the page, including external CSS and JavaScript files.
