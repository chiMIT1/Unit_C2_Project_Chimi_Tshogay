Lesson 9 : File Paths

File Paths

A File Path is the route you dictate to the computer in order to find a file.

In the example on the bottom, the absolute path of the project.docx file is C:\Users\Al\Documents\project.docx

If you were working in the Al folder, you could find project.docx's relative path by telling the computer Documents\project.docx

Ex:

When to use absolute and relative paths

It is best to use absolute paths when referring to files that exist outside of your current computer.

It is often best to use relative paths when you are referring to files that are on your current computer or server. This way, if the folder that contains the files is moved, the links between files does not break. Users that access the file remotely also have little problem linking the files.
